# Tabla Predictiva LL(1)

**Terminales en columnas (agrupados):**

- `ID` = identificadores (`ID`)
- `LIT` = literales (`NUM_INT`, `NUM_FLOAT`, `NUM_EXP`, `STRING`, `CHAR`, `TRUE`, `FALSE`)
- `PAREN_IZQ` = `(`
- `PAREN_DER` = `)`
- `OP_SUMA` = `+`
- `OP_RESTA` = `-`
- `OP_MULT` = `*`
- `OP_DIV` = `/`
- `OP_MOD` = `%`
- `COMA` = `,`
- `PUNTO_COMA` = `;`
- `LLAVE_IZQ` = `{`
- `LLAVE_DER` = `}`
- `IF`, `WHILE`, `FOR`, `SWITCH`, `RETURN`, `CLASS` = palabras reservadas
- `TIPO` = `{ INT, FLOAT, DOUBLE, CHAR, BOOL, VOID }`
- `OP_ASIG` = `=`
- `OP_NOT` = `!`
- `OP_IGUAL` = `==`
- `OP_DISTINTO` = `!=`
- `OP_REL` = `<, <=, >, >=`
- `OP_AND` = `&&`
- `OP_OR` = `||`
- `BOF`, `EOF`
